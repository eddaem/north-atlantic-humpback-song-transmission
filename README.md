# North Atlantic Humpback Whale Song Cultural Transmission (2008–2012)

## Overview

This repository contains the R scripts used in the analysis reported in:

> Magnúsdóttir, E.E., et al. (in press). Subarctic feeding grounds as cultural exchange hubs in the North Atlantic humpback whale song network. *Royal Society Open Science*. DOI: [to be added upon publication]

The study investigates cultural transmission of humpback whale (*Megaptera novaeangliae*) song between a subarctic feeding ground in Iceland and two tropical breeding grounds — the Lesser Antilles (Caribbean) and the Cape Verde Islands — across 2008–2012. Analyses include unit classification validation, Markov chain song sequence reconstruction, theme similarity (LSI and DSI), bootstrapped hierarchical clustering, and Shannon transition entropy.

## Running order

Scripts are numbered to indicate the recommended running order:

| Script | Description | Methods section |
|--------|-------------|-----------------|
| `01_unit_classification_RF.R` | Random Forest validation of human-assigned unit categories | §3.3 |
| `02_markov_sequences.R` | Markov chain transition analysis; set median theme strings | §3.4 |
| `03_LSI_similarity_dendrograms.R` | LSI pairwise similarity and pvclust dendrograms | §3.5 |
| `04_DSI_similarity_dendrograms.R` | DSI theme repertoire overlap and pvclust dendrograms | §3.6 |
| `05_transition_entropy.R` | Shannon transition entropy of theme sequences | §3.7 |

## Data

The `data/` folder contains the input files required to run the scripts. 

| File | Description |
|------|-------------|
| `Song_Unit_Frequency_data_combined_results.txt` | Acoustic measurements of song units across all location-years |
| `phrases_combined.csv` | Annotated theme sequences exported from Raven Pro 1.4 |
| `phrases.tsv` | Processed theme sequence data with unit strings |
| `set_median_theme_strings.tsv` | Set median unit strings per theme per location-year (output of script 02) |

## Key custom functions

| Function | Script | Description |
|----------|--------|-------------|
| `lv_unweighted()` | 03 | Unweighted Levenshtein distance |
| `run_pvclust_setmedian()` | 03 | pvclust bootstrapped dendrogram with AU/BP overlay |
| `dice_pair()` | 04 | Pairwise Dice's Similarity Index |
| `calc_entropy()` | 05 | Shannon entropy from transition probability vector |
| `reconstruct_sequence()` | 02 | Markov chain sequence reconstruction |
| `get_medoid_idx()` | 02 | Set median (medoid) identification |
